# Elliptic Secrets

## Deskripsi

Ini merupakan challenge dengan kategori _Cryptography_, tapi kenapa kok menggunakan file .pcap?<br>
Karena saya ingin memberikan challenge yang memang ada di _real life_, bukan challenge enkripsi _flag_ suka suka.
Selain itu saya juga ingin agar challenge ini dapat dikerjakan dengan kombinasi skill peserta dan AI dengan seimbang, bukan hanya berikan challenge ke AI lalu selesai. Maka dari itu, pada challenge ini saya menggunakan tema dekripsi pesan yang dikirim menggunakan protokol HTTPS yang memiliki _vulnerability_.
